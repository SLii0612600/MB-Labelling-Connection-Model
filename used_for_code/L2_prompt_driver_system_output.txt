## driver system output 下的可用L2标签

【reduced】 
定义：指驱动系统（含电驱/混动/燃油传动）在故障或保护状态下主动降低输出能力的情况，通常以仪表“小乌龟”“驱动性能降低/受限/性能严重受限”等提示表现，可导致加速变差、最高车速受限甚至短时无法行驶。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表出现小乌龟图标并提示“驱动性能降低/受限” 
- 车辆加速无力或最高车速被限制在40–120km/h区间 
- 行驶中突然动力下降或挂入N档，需熄火重启后暂时恢复 
- 故障多在电量剩余一定百分比或续航剩余某里程时触发 
- 充满电/油情况下续航里程明显变短且伴随驱动性能降低报警 
- 故障码指示驱动系统进入保护（如高压电池单体电压差大、电机/增压系统异常）导致功率受限
