## steering column switch module 下的可用L2标签

【broken】 
定义：转向柱开关模块“broken”问题，指雨刮/组合开关本体或其旋钮、塑料件等机械结构发生脱落、松脱、断裂等物理损坏，导致部件不能牢固固定或存在掉落风险。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 雨刮操作旋钮/按钮在扭动或操作过程中脱落 
- 轻轻一拉雨刮调节旋钮就从开关组件中脱离 
- 开关塑料件未卡牢、易松动并有再次掉落风险 
- 外观无明显损伤但结构配合不良导致脱落 
- 雨刮开关头部旋钮配合不良、装配不牢固 
- 开关脱落影响雨刮正常使用及高速行驶安全

【malfunction】 
定义：转向柱开关模块的功能/软硬件故障，导致换挡、转向灯、雨刮、方向盘按键及相关驾驶辅助/通讯功能异常，或在更换/升级过程中模块无法正常工作。 

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 启动车辆后无法挂挡或挂不上/挂不回P挡 
- 仪表提示“请勿换挡”、多系统故障灯全亮伴随无法换挡 
- 打转向灯时雨刮器自动乱刮或雨刮报警、无法关闭 
- 方向盘多功能按键、喇叭、安全气囊等因转向柱模块内部故障失效 
- 转向灯开关不回位、回正后错误侧转向灯闪烁 
- 更换新转向柱模块后无法通讯、无法试运行或持续报故障码 
- 升级/编程N80后转向角传感器信号异常、ESP/驾驶辅助功能受限 
- 转向柱模块内部电气/软件间歇性故障，偶发死机、雨刮乱动、锁车休眠后短暂恢复

【noise】
定义：指转向柱开关/控制模块在车辆转向或转向柱电动调节过程中发出的各种异常声音（如咔哒、嘎吱、滋滋等），包括模块内部机械结构、模块与方向盘/转向柱等周边部件摩擦或干涉所产生的噪音，但不涉及功能失效或操控跑偏等问题。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类）
- 原地或低速打方向时，方向盘下方传出咔哒/嘎嘎/咯吱异响 
- 转向至特定角度（如左转90°–180°、快打到接近打死位）出现持续或节奏性异响 
- 异响明确来自转向柱模块内部，类似塑料件摩擦、弹簧拧劲或发条拨动声 
- 方向盘与转向柱模块、气囊线束等接触面摩擦产生滋滋/呼呼/嘘嘘声 
- 冷车、热车或晒热后异响程度明显变化，但转向功能本身正常 
- 多次更换/润滑转向柱模块后异响反复出现，主要影响驾乘舒适性而非安全功能

## 分类原则（重要！）

- 只要反馈内容符合L2的定义描述，即使与列出的特征性话题不完全匹配，也应归入该L2
