## e mode 下的可用L2标签

【malfunction】 
定义：e mode 的故障类问题，指车辆在应具备电动/混动驱动功能的前提下，因系统硬件或软件异常导致电动模式/纯电行驶无法正常启用、持续或切换，而非因正常策略（如电量/温度限制）产生的性能差异。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 纯电模式/电动模式按钮为灰色或选项不可选 
- 电量充足但始终无法纯电行驶，只能燃油驱动 
- 行驶中电动模式被强制退出或突然切回燃油模式 
- 仪表提示“电动驾驶模式暂时不可用”并伴发动机故障灯亮 
- 冷天、极端温度或高海拔等环境下电动模式长期不可用且被认定为异常 
- 充满电后仍提示电动模式不可用或 EV 功能受限 
- 升级/编程（ME、BMS/PTCU 等）后 e mode 功能丢失或无法再启用 
- 伴随高压电池/接触器/高压控制单元故障码导致电动模式失效 
- 纯电热车或开空调时触发报错，电动模式关闭且空调功能受限
