## active front lighting system 下的可用L2标签

【malfunction】 
定义：指主动前照明/自适应灯光系统（含数字大灯、自适应远光辅助、智能照明系统等）因故障或控制错误，导致功能受限、无法启用或工作异常的情况，包括硬件与软件层面的失效。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表提示“数字大灯功能受限”或“自适应灯光系统故障” 
- 显示自适应远光/远光辅助“暂时不可用”“停止运作” 
- 自动远光灯从未点亮或仅保持近光，功能形同失效 
- 自动远近光无法自动切换需全程手动控制灯光 
- 智能照明/自动大灯偶尔失效，功能时好时坏 
- 诊断为大灯控制单元、灯光模块或传感器损坏需更换

【stop working】 
定义：指主动前照明/自适应远光灯/数字大灯等智能照明相关功能在行驶或使用过程中被系统关闭、失效或部分失效，并在仪表或车机上提示“自动/智能照明系统（模式）/自适应远光灯辅助系统/某侧大灯停止运作/停止运行/功能受限”等报错，导致该功能无法正常工作。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表提示“自动照明模式/智能照明系统停止运作” 
- 提示“自适应远光灯辅助系统停止运行/不可用” 
- 数字大灯功能受限或某侧智能大灯单侧停止运作 
- 自动/智能前照明功能行驶中偶发失效需到店检测 
- 停止运作伴随需更换照明控制单元或相关传感器 
- 新车或低里程阶段智能照明频繁报停影响夜间使用
