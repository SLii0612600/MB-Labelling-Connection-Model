## L1组件/功能说明

**coolant system** 是指冷却系统（coolant system）是为发动机及新能源车型的电驱/电池等部件提供冷却的系统，由冷却液、水泵、节温器、散热器、水管/软管、膨胀水壶、转换阀、循环泵及相关传感器/控制单元等组成，用于保持部件在安全工作温度并防止过热或冻裂。

## coolant system 下的可用L2标签

【broken】 
定义：冷却系统中软管、回水管、散热器等部件出现物理性断裂、裂开或破损，导致冷却液泄漏、水温/液位报警甚至影响车辆行驶或启动的结构损坏问题。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 行驶中冷却液软管/回水管突然断裂脱落 
- 冷却液管焊接处或管体本身裂开渗漏 
- 膨胀水壶（冷却液壶）连接水管断裂导致液位报警 
- 外力或老化导致冷却液管路物理破损 
- 冷却液散热器本体外观破损影响冷却功能 
- 管路断裂后机舱出现白烟/水雾并触发水温或加注冷却液报警

【leaks】 
定义：冷却系统“leaks”指冷却液（防冻液）从冷却回路各部件或连接处向外泄漏/渗漏，或内部窜入发动机、高压电池、电机等部件，导致液位降低、报警甚至高温或车辆抛锚的故障。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 地面、底护板/发动机舱出现粉红色或红色冷却液水迹 
- 仪表频繁出现“加注冷却液/冷却液液位低”报警伴液位下降 
- 水管、回水管、通风管等塑料管路老化开裂、脱落导致漏液 
- 水泵、电动水泵/48V冷却液泵接头或壳体渗漏冷却液 
- 节温器本体或密封垫、缸体水道等处密封不良造成冷却液渗漏 
- 散热器、低温冷却器、水箱或冷凝器等换热器本体/接口渗水 
- 冷却液从发动机内部泄漏（缸垫损坏、缸内/进气道进防冻液）引起缺液或抖动 
- 高压蓄电池、电机冷却回路泄漏防冻液，底部或透气孔有液体渗出 
- 冷却液管路因事故、机脚/支架断裂或外力挤压破损导致突然大量漏液 
- 新车或短期内多次出现冷却液亏损、频繁补加仍找得到渗漏痕迹

【malfunction】 
定义：冷却液/冷却系统的故障问题，指冷却回路中零部件或控制系统出现功能失效或性能异常，导致系统报警、水温异常、车辆性能受限甚至无法行驶，而非单纯液位正常消耗或用户误操作。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表显示“冷却系统故障/立即停车/关闭发动机”等红色报警 
- 行驶中水温迅速升高或水温灯亮，伴随限速、动力下降或熄火 
- 冷却液泵、电动水泵转速过低、卡死、通讯故障导致报警 
- 冷却系统管路/水管/通风管破裂、渗漏致冷却液大量流失 
- 冷却液节温器、转换阀、温度/液位传感器等元件失效引发故障码 
- 新车或短里程频繁出现冷却系统故障，多次维修/升级仍反复报警

【overheated】
定义：冷却系统“overheated”问题，指发动机或冷却液温度异常升高，达到或接近报警/红区，导致仪表报“冷却液温度高/关闭发动机”等高温警告，甚至影响甚至中断车辆正常行驶。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类）
- 仪表显示水温高到120–130℃并进入红区
- 出现“冷却液温度高/停车关闭发动机”红字报警
- 行驶途中突然水温高，需靠边停车等待拖车
- 水温快速上升、风扇长期高速运转仍温度不降
- 高速或城市正常行驶中反复出现高温报警
- 水温高同时出现动力下降、加速无力或无法继续行驶

【warning】 
定义：冷却系统的报警问题，指仪表或车机对冷却液温度/液位/冷却系统故障发出各类警告提示（文字、图标、黄灯或红灯），包括“加注冷却液”“冷却液系统故障”“水温高/发动机高温”等各级别报警，无论实际根因是缺液、泄漏、泵或阀故障等。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表提示“加注冷却液”“冷却液液位低”黄字/黄灯报警 
- 行驶或启动后出现“水温高/发动机高温”，伴随红色水温灯 
- 文字提示“冷却液，停车，关闭发动机”等禁止继续行驶警告 
- 冷却系统/冷却液故障灯常亮或频繁点亮，实际温度/液位有时正常 
- 冷却液报警与“驱动性能降低”“冷却系统故障”系统信息同时出现 
- 冷车启动即出现黄色水温灯/冷却液故障提示，短时间内难自行消除

## 分类原则（重要！）
