## coolant system hose 下的可用L2标签

【broken】 
定义：冷却系统水管/软硬管、接头或回水管等发生物理性破坏，导致冷却液/防冻液突然泄漏的结构损伤问题，包括爆裂、断裂、破损、接口脱落等情形，不包括单纯老化渗漏或设计材质争议。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 行驶中冷却液水管突然爆裂或崩开 
- 发动机舱冒烟、冷却液喷出或大量漏液 
- 仪表冷却液液位低/水温高报警后检查发现水管断裂 
- 新车或刚保养后短期内出现水管爆裂/断裂故障 
- 同一车辆多次发生不同位置冷却液水管爆裂抛锚 
- 冷却液管路接头、回水管等局部位置断裂或脱落

【cracked】 
定义：冷却液系统软管/水管出现开裂或断裂，导致冷却液渗漏或喷出，引发冷却液报警、水温过高甚至车辆无法行驶的问题。该类问题通常表现为软管本体或接头位置产生裂缝、爆管或断裂，而非仅仅是松脱或渗漏。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 冷却液水管/软管在行驶中突然爆裂导致高温报警 
- 发动机舱内冷却液软管接头处开裂漏防冻液 
- 防冻液管路因老化硬化后一压就断或多处裂纹 
- 高速行驶或长时间行驶中冷却系统水管断裂抛锚 
- 保养或短期使用后冷却液软管再次开裂需反复更换 
- 新车或使用年限较短即出现冷却液管路爆裂、裂缝问题
