## electrically retractable running board 下的可用L2标签

【malfunction】 
定义：电动伸缩踏板的故障问题，指踏板在应当自动伸出/收回或保持位置时未按设定动作、动作异常或结构部件失效，包括偶发和持续性的电子、电机、机构及安装相关失效。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 开门时踏板不弹出或四门都无动作 
- 踏板伸出一半即停止或又自动缩回 
- 踏板收不回去、锁车后仍停在伸出位置 
- 单侧踏板（多为右侧）长期或间歇性不工作 
- 仪表/中控提示“踏板未伸出/功能不可用” 
- 多次关门再开门后踏板才恢复动作 
- 电机卡滞、控制单元无通讯但线路电源正常 
- 踏板掉落、支架断裂或侧边板/下饰板松动变形 
- 步上踏板时突然回缩，存在夹脚或摔落风险 
- 新车或加装后即频繁出现踏板失灵、需反复学习/升级仍未解决
