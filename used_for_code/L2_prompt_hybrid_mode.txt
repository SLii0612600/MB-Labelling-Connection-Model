## hybrid mode 下的可用L2标签

【shift roughly】 
定义：混动/油电模式切换过程中的换挡粗糙问题，指电驱到发动机驱动或发动机到电驱切换时伴随明显冲击、顿挫、抖动等不平顺传动感受（通常伴有“咔/咚/哐当”等机械声）。 

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 纯电切换燃油瞬间车身明显顿挫或前后闯动 
- 电驱切换发动机时有强烈冲击感并伴随“咔/咚/哐当”声 
- 电量耗尽或电池电量模式下发动机介入时单次剧烈抖动 
- 低速加油门触发发动机启动，出现一次性冲击/耸动 
- 仅油电切换瞬间有冲击和异响，纯电或纯油行驶过程平顺 
- 热车或冬季低温等特定环境下油电切换顿挫现象更明显

【shifts roughly】 
定义：指车辆在混动模式下由电机驱动切换至内燃机驱动时，动力衔接不平顺，伴随明显冲击、顿挫或拖拽感的换挡/切换问题，有时同时伴随底盘异响。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 纯电切换至发动机时车身明显顿挫、冲击 
- 电转油瞬间底盘“咯噔”或“呲呲”异响伴随抖动 
- 起步缓踩油门、电切油时有拖拽感与冲击感 
- 低速约18km/h电驱动切换到发动机驱动时顿挫 
- 电、油模式单独行驶正常，仅切换瞬间冲击不适 
- 频繁市区走走停停时电转油切换影响乘坐舒适性
