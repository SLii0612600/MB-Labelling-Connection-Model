**instrument cluster** 是指Instrument cluster（组合仪表/仪表盘），指位于驾驶员前方、集成车速、转速、警示灯、行车信息等的显示与控制总成，当前多为液晶/全液晶显示屏并由仪表控制单元（如N133/1）驱动。

## instrument cluster 下的可用L2标签

【flickering】 
定义：指组合仪表/驾驶员显示屏在工作过程中出现间歇性闪烁、跳闪或黑一下又亮的现象，表现为画面不稳定而非持续性花屏、白边或固定线条类缺陷。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 行驶中整块仪表屏幕一眨一眨闪烁 
- 冷车首次启动后短时间内出现闪屏现象 
- 仪表画面频繁在正常显示与黑屏之间切换 
- 熄火锁车再启动车辆后短时间内反复闪屏 
- 某些操作或界面切换瞬间屏幕闪一下/黑一下 
- 闪屏同时伴随仪表类似“重启”再恢复显示

【flikering】
定义：仪表显示屏的闪屏/花屏问题，指仪表屏画面出现闪烁、雪花、条纹、花纹等不稳定或异常显示现象，包括间歇黑屏后又恢复等。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类）
- 行驶途中仪表屏一闪一闪、间歇黑屏又恢复
- 画面出现绿色竖纹、模糊条纹、雪花点状杂波
- 屏幕出现固定花纹或一道纹路，仅亮屏时可见
- 仪表屏偶发重启伴随短暂黑屏或花屏
- 事故后拆装仪表台后出现持续花屏/条纹
- 故障时无相关故障码，屏幕电气/内部问题怀疑较多

【inconsistent torque display】 
定义：仪表盘及中控屏上显示的发动机扭矩/功率数值与车辆官方标称或车辆实际性能不一致的问题，通常由软件或控制模块参数错误导致。该问题聚焦于“显示与应有数值不符”，而非发动机本身的机械故障。 

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- E300/E300L车机和仪表盘显示320Nm、150kW等E260参数 
- 屏显扭矩功率与合格证、铭牌或官网标注400Nm、190kW不一致 
- 仅7–9月等特定批次车辆出现扭矩显示为低功率版本现象 
- 官方或经销商说明为软件/模块（如PTCU、N127）数据偏差需升级修复 
- 升级前后屏幕数值改变（由320变400），用户质疑是否仅“改数字” 
- 通过读取控制模块版本号发现与正常E300车型不一致，引发对显示正确性的质疑

【malfunction】 
定义：指组合仪表/仪表显示屏本身软硬件或电气系统出现故障，导致显示内容异常、功能缺失、死机/黑屏或无法正常通信等影响仪表正常工作的情况，不包含纯外观瑕疵或装配缝隙问题。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 熄火锁车后仪表不休眠、屏幕常亮或奔驰Logo长时间不灭 
- 仪表黑屏、白屏、灰屏或部分区域不显示内容 
- 总里程/里程、小计、油耗等数据长期为虚线或完全不显示 
- 仪表显示信息不全，仅剩车速/档位/时间等少量字段 
- 导航、收音机、能量流等菜单在仪表内一直“加载中”无法显示 
- OTA或店内升级仪表软件后，仪表控制单元死机、无法通信或一直卡在升级界面 
- 行驶中仪表出现乱码、异常红色英文提示或多故障灯同时点亮 
- 仪表盘提示“驱动性能降低”“请勿换挡直接驶向授权服务中心”等系统故障信息 
- 启动后转速表/车速表指针异常归零、冻结或与实际状态不符 
- 仪表控制单元（IC/N133等）在快速测试中通讯缺失或多次报内部电气故障

【milky border】 
定义：指液晶仪表显示屏在边缘或局部出现持续的发白/雾状白边、白色印记或光晕，与正常显示内容无关，多由屏幕内部脱胶、漏光、液体/异物腐蚀等造成的外观缺陷。 

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表屏上部或四周出现一圈雾状白边 
- 自然光或阳光下边框发白、泛光尤为明显 
- 熄火黑屏状态下白边更清楚，可擦拭但无法去除 
- 白边呈条状/波浪状/不规则云雾状，常在边角或边缘内侧 
- 伴随描述为“脱胶”“开胶”“漏光”“有气泡/空气层”等内部结构问题 
- 说明为不影响功能但严重影响观感、美观或使用体验

【milky display】 
定义：指组合仪表液晶显示屏出现泛白、雾状、白边、漏光等“发白/泛雾”现象，导致屏幕局部或边缘亮度、颜色异常，但多数情况下功能显示仍正常。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 屏幕上部或四周长期“发白”“漏白光” 
- 显示屏边缘/角落出现固定白边条纹 
- 屏幕内侧有雾状白印、白道或不规则雾纹 
- 屏幕内部小白点、白色印记擦拭不掉 
- 自然光或点亮/黑屏状态下均能看到发白区域 
- 疑似内部脱胶、漏光或水汽进入导致的泛白现象

【no display/black/blank/blue screen】 
定义：指组合仪表/驾驶员显示屏在应当点亮时完全无画面显示，呈黑屏、空白屏或异常纯色屏（如白边、白屏、蓝屏），包括短暂黑屏后自动恢复和持续黑屏均归入此类，不涉及内容错误或单个指示灯异常。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 行驶过程中仪表突然整体黑屏或几秒后自动恢复 
- 启动车辆后仪表长期不亮，仅剩档位或里程等少量字符 
- 仪表与中控屏同时黑屏或仅仪表黑屏、娱乐屏正常 
- 黑屏状态下车辆可行驶但无法看到车速油量等关键信息 
- 黑屏偶发与频繁并存，常通过锁车休眠或断电后短暂恢复 
- 新车或低里程阶段即出现持续或反复的仪表黑屏/白边白屏

【noise】 
定义：仪表板/仪表盘及其内部部件在工作或车辆行驶过程中产生的各类异常声响问题，包括摩擦声、敲击声、电流/风扇嗡嗡声等，影响车内静谧性与乘坐舒适性，但通常不直接影响功能。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 颠簸或不平路面行驶时仪表台内哒哒/咯吱/格叽异响 
- 冷车启动或刚解锁时仪表台内持续嗡嗡风扇声几分钟 
- 仪表屏幕后方或饰板松动引起的塑料件碰撞、抖动声 
- 仪表区出现类似“电流声/耳鸣声/滋滋声”的高频噪音 
- 特定操作触发的声响：打方向、轻点油门、踩刹车瞬间滴答/咔哒声 
- 仪表台内部传感器、风门电机、微粒/微尘传感器等工作时异常杂音

【pixel defect】 
定义：仪表显示屏的像素/显示缺陷问题，指液晶屏局部像素或显示区域异常，如出现线条、白边、花屏、水波纹、缺块等，虽多不影响功能运行，但影响读数完整性和观感。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 屏幕中间或左右出现单条竖线/横线 
- 局部多条竖纹或整屏花屏导致内容难以辨认 
- 显示区域边缘出现规则或不规则白边、空白边 
- 屏幕边框或边缘泛白、水波纹等局部亮斑现象 
- 屏幕上方未显示至边界，存在明显未点亮区域 
- 线条、白边在阳光或强光下更明显但功能正常

【scratching】 
定义：仪表显示屏或仪表台表面出现划痕、裂纹、印痕等机械性损伤的问题，包括轻微细划、明显裂缝、自裂或疑似外力造成的开裂、波纹印痕等，不涉及显示内容本身功能异常。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表屏表面有明显裂痕或开裂 
- 屏幕仅在熄屏或强光/灯光照射下可见划痕 
- 屏幕局部出现白色线条或白色印痕似裂纹 
- 显示屏上方/边缘出现波浪状纹路印痕 
- 新车或更换后仪表屏自带细小划伤 
- 客户自述未受撞击却发现屏幕自行开裂

【warning】 
定义：仪表屏/组合仪表的告警与故障提示问题，指仪表盘上异常点亮或反复出现各种故障灯、报警信息（含文字/图标），与底层硬件损坏、通讯中断或整车其它系统真实故障无关的显示与告警行为本身。 

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 行驶或启动时仪表盘突然“故障灯全亮 / 多个故障灯齐亮” 
- 仪表反复弹出系统故障、辅助功能停止运作等报警提示 
- 仪表出现红色严重警告信息（如“请勿行驶”“请勿熄火直接驶向服务中心”） 
- 仪表显示最高车速被限制（如80km/h）或“性能受限”类警告 
- 仪表提示“无法显示故障信息，请联系维修车间”等英文红色信息 
- 故障报警偶发出现，熄火重启后短暂消失又复现，车间难以重现 
- 仪表报智能辅助、驾驶辅助、灯光、SRS、安全带等子系统故障/受限 
- 仪表异常报警伴随档位不显示、提示“请勿换挡驶向授权中心”等信息

【water mark/unusual spot】 
定义：仪表盘液晶显示屏或其覆盖玻璃/外屏内部出现水印、白边、白斑、黑点、绿点、气泡等异常斑点或印迹的外观缺陷问题，包括脱胶、漏液等引起的局部异常痕迹，但通常不影响仪表功能显示。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 屏幕内部固定白点/黑点/绿点，擦拭表面无法去除 
- 熄屏或低亮度时更明显的白斑、阴影、雾状印记 
- 屏幕四角或边缘一圈白边／白色条纹，类似漏液或脱胶水印 
- 仪表中部大片白色点点或类似贴膜气泡的斑点 
- 屏幕局部气泡状圆圈或鼓起斑点，仅在特定光线或角度可见 
- 仪表内屏出现水印/雾气/磨砂状发白痕迹，主要影响美观

【wrinkle/dent】 
定义：仪表台或液晶仪表/显示屏周边饰板的表面出现皱纹、凹坑、凹陷、波浪纹等形变，或表皮/边框贴合不良导致的局部不平整、折痕、小窝等外观缺陷（含轻微开胶引起的形变），不涉及功能失效。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表台表面局部凹陷/小窝，摸起来较软 
- 副驾仪表台台面大面积凹凸不平、坑坑洼洼 
- 仪表台皮革高低起伏不顺，有折痕或波浪纹 
- 液晶仪表屏幕边框不贴合，边框翘起留缝 
- 仪表盘或仪表台出现一条直线状凹痕/印子 
- 仪表台表皮与内部骨架分层，按压有空鼓感

## 分类原则（重要！）
- **以定义为准**：判断反馈是否属于某个L2时，以该L2的"定义"为核心依据
- **特征性话题是示例**：每个L2下列出的"特征性话题"是该L2的常见情况示例，不是完整列表
- 只要反馈内容符合L2的定义描述，即使与列出的特征性话题不完全匹配，也应归入该L2
