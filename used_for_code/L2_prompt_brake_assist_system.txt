## L1组件/功能说明

**brake assist system** 是指Brake Assist System / Active Brake Assist（制动辅助系统/主动式制动辅助）。通过雷达、摄像头等监测前方车辆和障碍物，在驾驶员制动不足或无制动时自动增强制动力或主动刹车，以缩短刹车距离并避免/减轻碰撞。
