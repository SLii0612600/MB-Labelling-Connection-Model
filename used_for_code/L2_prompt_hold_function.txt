## L1组件/功能说明

**hold function** 是指HOLD / Auto HOLD（自动驻车功能），通过ESP/制动系统在车辆完全停稳后自动维持制动力，使驾驶员松开制动踏板车辆仍保持静止，多用于等红绿灯、缓行路况和坡道起步等场景。

## hold function 下的可用L2标签

【current alike noise】 
定义：指车辆启用或保持HOLD（自动驻车）功能时，从制动系统/车轮处传出类似电流声、咯噔声、咚咚声等异常噪音的现象，通常与ESP/制动液压控制过程有关，但制动功能本身多保持正常。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- HOLD启用静止状态下周期性“咯噔/咚咚”声 
- 等红绿灯开启HOLD后过数秒或数十秒出现单次咯噔声 
- HOLD保持过程中车底或车轮处传来间歇敲击样噪音 
- 启用HOLD后伴随轻微顿挫感或轻微车身前窜同时有异响 
- 制动器或车轮区域出现高频电流样嗡鸣/滋滋声 
- 噪音可在原地反复重现且与踩刹/激活HOLD动作直接相关

【malfunction】 
定义：HOLD自动驻车功能或驻车系统的故障失效问题，指在应当保持/释放车辆制动或提供驻车保持时，功能无法正常启用、维持或退出，出现溜车、抱死、故障报警等异常情况。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 深踩刹车仪表不显示HOLD字样，功能始终无法启用 
- HOLD偶发或持续不起作用，需多次重踩刹车才可能激活 
- HOLD已显示启用但车辆仍缓慢前移或明显溜车 
- HOLD状态下车辆前窜/闯动，并伴随“咯噔”“哒哒”异响 
- HOLD自动解除，刹车踏板下沉，车辆自行向前滑行 
- 启用HOLD后起步踩油门车辆不走或起步熄火 
- HOLD功能与ESP/制动系统/驻车系统故障灯同时报警停用 
- 更换蓄电池传感器、执行软件升级后出现HOLD功能失效 
- 驻车系统锁死或驻车器无法释放，车辆无法起步或行驶 
- 仪表频繁提示“驻车系统故障/驻车制动器故障”，需更换驻车模块/电机/控制单元

【noise】 
定义：指与HOLD自动驻车功能相关的各类异常声响问题，包括HOLD启用、保持和释放过程中，来自车轮制动器、刹车系统或踏板等位置的异响或异常噪音（如咯噔声、嗒嗒声、呲/呼/泄气声等），而车辆制动是否有效可有可无。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- HOLD启用后车轮处周期性“咯噔/噔噔/咔咔”异响 
- 等红绿灯或长时间HOLD时，每隔数秒伴随轻微晃动的敲击声 
- HOLD保持一段时间后突然“噔一下”或连续金属敲击感异响 
- 解除/起步释放HOLD时刹车踏板位置出现“咕”声或弹簧声 
- HOLD启用或松开制动时出现明显“呼/嗤嗤”放气类噪音 
- 安静环境下D挡HOLD驻车时可清晰听到来自前后轮制动器的局部噪音

【shaking】 
定义：指在HOLD功能启用或保持状态下，车辆出现异常抖动、顿挫或冲击感的现象，包括车身、踏板或车内可感知的震动，而正常行驶或未启用HOLD时不存在或明显减轻。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 等红灯启用HOLD后车身间歇性抖一下 
- D/R挡驻车且HOLD激活时车辆一冲一冲向前顶 
- HOLD保持时伴随“咯噔/咯咯”声与轻微冲击感 
- 踩出HOLD瞬间出现连续轻微顿挫或晃动 
- 启停配合HOLD熄火后前舱传来振动感传至踏板 
- 未挂N档仅D档+HOLD时座椅方向盘有明显轻微抖动

【stop working】 
定义：指驻车/HOLD/驻车辅助系统在应正常运行时出现完全或间歇性停止工作，仪表提示“系统停止运作/不可用”等，导致相关自动驻车、雷达、影像等功能无法使用的问题。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表报警“驻车系统/驻车辅助系统停止运作” 
- 显示“驻车辅助系统和PARKTRONIC停止运作/不可用” 
- HOLD功能偶发失灵，松刹车后车辆自行溜车 
- 驻车系统失效导致倒车影像、雷达同时不可用 
- 行驶中偶发短暂停车2–3秒后又自行恢复 
- 系统重启或升级后暂时恢复，故障间歇反复出现
