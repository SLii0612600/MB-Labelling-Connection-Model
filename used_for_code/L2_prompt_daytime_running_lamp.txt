## daytime running lamp 下的可用L2标签

【malfunction】 
定义：日间行车灯的功能/部件出现故障或失效，导致其不能正常工作或状态异常，包括不亮、变暗、变色或仪表持续报“日间行车灯故障”等问题。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 单侧或双侧日间行车灯不亮、时亮时不亮 
- 仪表提示“日间行车灯×侧故障/报警” 
- 日间行车灯明显发黄、变暗，与另一侧颜色/亮度不一致 
- 日间行车灯灯带、LED光源烧毁或内部断路 
- 需更换整个大灯总成或大灯模块才能恢复日行灯功能 
- 日间行车灯长期常亮后损坏，或刚过保即出现日行灯失效

【one lamp off】 
定义：日间行车灯“单侧不亮”问题，指车辆左右日间行车灯中只有一侧失效（熄灭或不发光），通常伴随仪表故障提示，但另一侧仍正常工作。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 左或右前日间行车灯单侧不亮 
- 仪表提示“日间行车灯/驻车灯光故障”但仅一边失效 
- 大灯总成一侧日行灯LED损坏需整体更换 
- 更换或检修后另一侧日行灯在短期内相继熄灭 
- 更换电瓶或拆装大灯后出现一边日行灯不亮 
- 行驶中发现一侧日行灯灭、另一侧仍正常发光
