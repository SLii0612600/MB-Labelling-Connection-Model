## door handle 下的可用L2标签

【can't eject/retract】 
定义：隐藏式车门拉手的伸出/缩回机构失效，表现为在解锁或锁车时拉手无法正常弹出、完全弹出，或无法完全缩回/回位（包括卡在中间、仅一端动作、需用力拉动等），从而影响正常开关门或拉手隐藏功能。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 解锁后单个或多个门把手完全不弹出，需从车内开门 
- 锁车后门把手无法缩回或只缩回一半，一端翘起 
- 拉手卡在中间位置，既伸不出来也收不回去 
- 仅某门把手长期或间歇性不随其他门一起伸缩 
- 需用较大外力/多次操作才能让把手弹出或回收 
- 拉手伸缩过程中与门框干涉、发卡导致动作不顺畅

【loose】 
定义：door handle 的松动问题，指门把手或其固定结构存在明显间隙、晃动或固定不牢，导致触感松垮，且可能影响正常开关门或把手伸缩动作的可靠性。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 门把手有明显间隙，轻按会晃动或下陷 
- 把手松旷导致用力拉时无支撑感、发空感 
- 松动后无法顺利打开车门或影响关门手感 
- 隐藏式门把手伸出后晃动，不够牢固 
- 松动部位与门板/托架干涉摩擦，伴随异响 
- 门把手固定螺栓、轴承托架等连接处松旷或损伤

【malfunction】 
定义：车门把手的故障问题，指门把手本体或其关联结构/电控单元工作不正常，导致开关门、伸缩、感应解锁/上锁或迎宾等功能异常，包含机械损坏、电气/软件故障等各类失效形态。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 隐藏式门把手无法伸出或缩回、伸出时卡滞 
- 拉动外把手/内把手需用很大力或多次拉动才能开门 
- 某一车门把手便捷进入/触摸解锁上锁无反应，其余车门正常 
- 解锁后个别门把手不弹出或锁车时把手无法收回 
- 把手轴承托架、支架、拉杆等内部结构断裂/损坏导致把手松旷、发卡 
- 把手控制单元/传感器报故障码或需频繁断电复位、刷机后短暂恢复又复发 
- 下雨、洗车或低温后门把手失灵、无法开关门或自动弹出 
- 门把手在锁车后自动再次弹出，或偶发需要多次操作才能锁止车辆

【noise】 
定义：门把手噪音问题，指在解锁/锁车、伸缩弹出、拉动开关门或回位过程中，门把手及其托架、胶条等部位发出异常声响（如吱吱、咯噔、咔咔等），影响使用感受或被认为不正常。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 隐藏式门把手伸出或缩回时出现吱吱/咯噔异响 
- 解锁或锁车时门把手位置处伴随明显滋滋/碰撞声 
- 拉动门把手开门时有咯吱、嘎吱或摩擦异响 
- 门把手回弹、回缩时声音过大，甚至让人受惊 
- 异响多出现在冷车、停车一夜后首次伸缩门把手 
- 托架、胶条、密封圈变形或润滑不良导致门把手摩擦异响

【stuck】 
定义：门把手“stuck”问题是指外门把手/隐藏式门把手在伸出、回收或手动拉动过程中发生卡滞、发卡或完全卡住，动作不顺畅甚至停止在中间或伸出位置，影响正常开关门或把手自动伸缩功能。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 解锁时门把手弹不出来或只弹出一部分 
- 锁车后门把手无法缩回，常停留在伸出/中间位置 
- 把手伸缩过程卡顿、发卡，需要用很大力才能开门 
- 门把手与门板/门框剐蹭导致油漆磨损、掉漆 
- 偶发或持续性卡滞，伴随把手晃动、抖动或“砰”一声弹开 
- 某一侧动作不同步（前后两端不齐）致把手倾斜并卡住

【worn】 
定义：车门把手及其装饰/密封件的表面或材质出现异常磨损、腐蚀、掉漆、起皮、划伤等老化损坏现象，影响外观或触感，但通常不影响开关门功能。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 内外门把手表面电镀层腐蚀、生锈起斑点 
- 门把手镀铬饰条起皮、边缘变锋利刮手 
- 皮质或喷漆门把手表面严重磨损、掉漆 
- 隐藏式门把手弹出位置存在明显摩擦划痕 
- 门把手内部橡胶/胶条被伸缩或摩擦磨损 
- 单个门把手相较其他车门出现异常加速磨损

## 分类原则（重要！）
- **以定义为准**：判断反馈是否属于某个L2时，以该L2的"定义"为核心依据
- **特征性话题是示例**：每个L2下列出的"特征性话题"是该L2的常见情况示例，不是完整列表
- 只要反馈内容符合L2的定义描述，即使与列出的特征性话题不完全匹配，也应归入该L2
