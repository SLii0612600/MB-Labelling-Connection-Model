## L1组件/功能说明

**key remote control** 是指遥控钥匙 / 无钥匙进入与启动系统的车钥匙组件，包含机械钥匙、电池、电路板及与车身天线、电子点火开关（EIS/EZS、N73/3、N69/5 等）配合，实现远程解锁/落锁、无钥匙进入（KG）、一键启动及应急启动。

## key remote control 下的可用L2标签

【hesitation】 
定义：指车辆感应钥匙或遥控钥匙在正常操作下反应滞后、需多次操作或等待一段时间后才完成解锁/开锁等指令的迟滞问题，而非完全无反应或功能失效。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 靠近车辆门把手已弹出但解锁延迟数秒 
- 车门把手弹出后立即拉门未解锁、需等待再拉 
- 停放一夜后首次解锁需按遥控键两三次才有反应 
- 需走到非常靠近门把手感应区才能触发解锁 
- 感应钥匙有时短时无反应，稍等片刻又可正常解锁 
- 同一路况下备用钥匙也存在按多次才响应的迟缓情况

【malfunction】 
定义：指遥控钥匙/机械钥匙/数字钥匙等与整车防盗启动系统之间的功能故障，表现为钥匙本身或其与车辆通信异常，导致解锁、上锁或启动车辆等基本功能失效或严重异常（含新配钥匙/订购钥匙无法正常使用）。 

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 一把钥匙失效，另一把正常（仅该钥匙无法解锁或启动车辆） 
- 遥控按键有灯亮但车辆无反应、需用机械钥匙开门 
- 仪表频繁/持续提示“检测不到钥匙”或“将钥匙放在指定位置” 
- 钥匙需放在应急启动区才能偶尔识别、正常随身携带无法启动车辆 
- 新订购或新配的钥匙可解锁却无法启动车辆或无KG无钥匙进入功能 
- 钥匙内部电子/芯片故障、进水/受外力后出现完全或间歇性失灵 
- 钥匙电池异常亏电或短期多次更换仍失效，被认定为钥匙本体故障 
- 机械钥匙或紧急钥匙无法开启车门、机械锁芯卡滞或损坏 
- 车辆可远程/数字钥匙正常，但物理遥控钥匙长期无法被识别或启动车辆

【warning】 
定义：钥匙遥控系统的报警/提示问题，指车辆在钥匙存在或使用过程中，仪表/中控关于“钥匙检测不到/钥匙报警/将钥匙放到指定位置”等相关警告信息出现异常、频繁或令用户困惑的情况。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表频繁提示“检测不到钥匙”但钥匙实际在车内 
- 启动车辆时出现钥匙信号丢失或需放到指定位置提示 
- 车辆打着火后钥匙离开，报警提示逻辑异常或提示几次后消失 
- 遥控钥匙无明显故障时仍出现钥匙报警信息 
- 用户不清楚钥匙指定放置位置导致报警提示难以消除

## 分类原则（重要！）
- **以定义为准**：判断反馈是否属于某个L2时，以该L2的"定义"为核心依据
- **特征性话题是示例**：每个L2下列出的"特征性话题"是该L2的常见情况示例，不是完整列表
- 只要反馈内容符合L2的定义描述，即使与列出的特征性话题不完全匹配，也应归入该L2
