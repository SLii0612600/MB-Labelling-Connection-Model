## L1组件/功能说明

**start/stop function** 是指Start/Stop function（自动启停功能），亦称 ECO 启停/自动节能启停。车辆在短暂停车（如等红灯）时自动熄火，在驾驶员准备起步时自动重新启动发动机，以降低油耗和排放。系统由发动机控制单元、变速箱控制单元、12V/48V电池及传感器共同控制。

## start/stop function 下的可用L2标签

【can't close/hard to close】 
定义：指车辆自动启停功能用户主观上想要关闭或永久关闭时，因系统设计或控制逻辑限制而无法关闭、无法设为默认关闭，或在某些工况下按键/菜单难以实现关闭的情况。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类）
- 自动启停无法设置为默认关闭，每次启动车都需手动关闭 
- 车辆设计不支持永久关闭自动启停，用户强烈希望彻底关闭 
- 冷车启动阶段按下启停按键无效，需踩刹车等特殊操作才能关闭 
- 启停功能原本上车都会先关闭，后来出现无法再开启/关闭异常 
- 用户咨询4S店被告知启停系统不能物理关闭或软件层面锁死

【can't start/hard to start】 
定义：指在启停功能介入后发动机自动熄火，但应重新启动时发动机无法/很难再次起动，需要多次操作或特殊步骤才能重新启动车辆的故障。该问题特指启停工作场景下的“再起动困难/不起动”，不包含首次点火启动车辆的问题。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 等红灯启停熄火后，松刹车或踩油门发动机不自动启动 
- 启停介入后挂到N档、车辆溜车或无法起步，需要重新点火 
- 需要按启动按钮多次或锁车等待一段时间后才能再次启动 
- 启停熄火后再次启动时起动机无动作、仪表灯全亮但不起动 
- 启停熄火后必须完全关闭点火/重启系统才能恢复行驶 
- 启停场景下偶发多次连续打火才能着车的情况

【malfunction】 
定义：启停功能的“malfunction”指发动机自动启停系统在应当工作时不能正常启停，或在启停过程中出现异常行为（如无法重启、失去动力、错误停机、功能长期被限制/不可用等）的故障类问题，而非单纯舒适性或设计逻辑上的正常现象。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 等红绿灯启停熄火后，踩油门发动机不重启或车辆不走 
- 启停触发后需挂P挡、重新点火才能再起步 
- 自动启停后短时间内车辆无动力输出或明显延迟响应 
- ECO指示长期黄色/白色，系统判定“ECO不可用”且始终不进入启停 
- 长期或间歇性“启停功能无法使用/不起作用”，与电瓶、软件或控制单元故障相关 
- 启停自启停熄火后直接进入熄火状态，多故障灯亮，需要重新启动车辆 
- 启停功能曾正常工作，保养或软件更新后出现无法启用或频繁失效 
- 自定义/菜单中关闭启停后，重启车辆设定无法被记忆，表现为启停控制逻辑异常 
- 启停触发后起动机启动困难，出现延时、咔咔异响甚至点火失败 
- Mercedes me 等后台提示电瓶/启停相关故障导致系统功能受限或失效

【noise】 
定义：指自动启停/ECO功能介入或退出时，车辆底盘、变速箱或发动机相关部位出现异常声响（如哒哒、铛铛、嗡嗡、噗噗等），与启停动作同步且令用户感到不正常或不舒适的声音问题。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- ECO/启停介入后底盘传来连续“哒哒哒”声 
- 启停熄火后挂P档出现“啪啪/铛铛”异响 
- 车身中部或变速箱区域有规律敲击声伴随启停 
- 启停工作时能听到变速箱辅助油泵明显运转声 
- 启停熄火后短暂“嗡嗡”电机声并伴随轻微震动 
- 等红绿灯启停动作时伴随“噗噗”等异常声音

【stop working】 
定义：指发动机自动启停功能在应当介入或恢复时不再工作，包括完全失效、偶发失效或启停过程后车辆无法自动重新启动等功能中断现象。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 等红灯踩刹车时启停不介入，发动机不熄火 
- 自动熄火后无法自动重新启动，车辆直接熄火瘫痪 
- 启停功能长时间几乎从不工作，仅偶发介入 
- 仪表显示“自动启停不可用”或黄灯提示，功能退出 
- 车辆长时间停放或电瓶亏电后启停持续失效 
- 新车或保养后即出现启停功能整体不工作

【warning】 
定义：启停功能的报警/告警问题，指与自动启停系统相关的黄色或红色提示灯、故障图标或文字警告反复点亮或常亮，无论启停功能此时是否真正工作失效。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 自动启停指示灯长期黄灯亮提示不可用 
- 启停故障标识/报警频繁出现又自行消失 
- 仪表同时出现启停图标与12V电瓶故障警告 
- 熄火/等红灯时启停失效并伴随黄灯提示 
- 启停报警与ABS/ESP等多系统故障灯同时亮 
- 关联操作（如开启360影像）后启停故障标识再次出现

## 分类原则（重要！）
- **以定义为准**：判断反馈是否属于某个L2时，以该L2的"定义"为核心依据
- **特征性话题是示例**：每个L2下列出的"特征性话题"是该L2的常见情况示例，不是完整列表
- 只要反馈内容符合L2的定义描述，即使与列出的特征性话题不完全匹配，也应归入该L2
