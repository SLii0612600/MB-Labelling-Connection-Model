## L1组件/功能说明

**instrument cluster control unit** 是指Instrument Cluster Control Unit（仪表控制单元/仪表盘控制模块），奔驰内部常记为 N133/1 或 IC。负责驱动仪表显示屏，处理并显示车速、转速、油量、挡位、故障灯及多种驾驶与安全信息，同时与车辆各控制单元进行通信。

## instrument cluster control unit 下的可用L2标签

【failed to upgrade】 
定义：指对仪表控制单元（N133/1 等）进行召回/服务措施或常规软件升级时，编程过程未能正常完成，导致升级任务失败的情况，常伴随控制单元进入加载/bootloader 模式、失去通讯或被“刷死”。 

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 执行召回/服务措施5492317等软件升级过程中报错或中断 
- 升级进度卡住、死机后提示升级失败或POST.FLASH错误码 
- 升级失败后仪表控制单元/N133/1无法通讯或无法被诊断读取 
- 升级失败后仪表处于加载模式/bootloader mode无法恢复 
- 诊断仪重启、多次重试或断电复位仍无法完成编程升级 
- 升级失败导致仪表黑屏、无显示并需更换仪表控制单元救回

【malfunction】 
定义：仪表盘控制单元/仪表控制模块的功能故障，表现为仪表黑屏、指示灯异常、显示内容/界面异常切换、无法通信等导致仪表显示或相关功能工作异常的问题。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 仪表显示屏黑屏或不亮但车辆可正常启动行驶 
- 仪表指示灯或报警灯成片闪烁、乱跳 
- 仪表盘控制模块烧毁、内部故障需整体更换 
- 仪表控制单元无法通讯导致多系统同时报故障 
- 启动后仪表显示风格或界面设置被自动重置/切换 
- 锁车休眠后仪表默认跳回时间日期界面而非用户上次界面
