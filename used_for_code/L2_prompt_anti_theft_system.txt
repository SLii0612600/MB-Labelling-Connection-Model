## L1组件/功能说明

**anti theft system** 是指Anti-theft system（防盗系统/主机防盗保护），是针对车载娱乐主机/MBUX/COMAND等多媒体控制单元的电子防盗功能。通过与整车编码、车辆识别信息及后台授权匹配，防止主机被非法更换、篡改或盗用，一旦判定异常即锁止多媒体功能，并在屏幕上显示“防盗保护已启用”等提示。

## anti theft system 下的可用L2标签

【active unexpectly】 
定义：防盗系统意外启用问题，指在未发生真实盗抢或未进行正常授权操作的情况下，防盗保护功能被错误激活，导致主机/娱乐系统或相关功能被锁止、提示“防盗保护已启用/防盗系统已启用”等信息。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类）
- 正常启动车辆或行驶中突然弹出“防盗保护已启用”提示 
- 未有加装改装且无盗抢行为，防盗功能自行触发 
- 防盗启用后中控/娱乐屏锁死，仅空调等少数功能可用 
- 防盗提示界面常驻，无法切换菜单或开启主机 
- 偶发或持续出现防盗启用提示，熄火重启/锁车休眠后仍存在 
- 远程升级、错误多次输入PIN码后触发防盗锁止，系统不能操作
