## L1组件/功能说明

**usb port** 是指车载 USB 接口（含 USB‑A、USB‑C/Type‑C），用于为外接设备（手机、平板等）提供电源充电，并作为多媒体接口传输数据（U盘音乐、行车记录仪、CarPlay/Android Auto 等）并与多媒体主机/多媒体连接单元（如 A90/5）通讯。

## usb port 下的可用L2标签

【can't connect】 
定义：USB 接口无法与外部设备建立正常连接的问题，包括无法识别设备、无法进行数据交互或由此导致无法充电等（前提是根源在接口/连接本身，而非设备内容或播放功能设置）。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 单个或部分 USB/Type‑C 口完全无反应、不能用 
- 插入手机/行车记录仪/U盘后系统不识别设备 
- 显示“无装置”或一直转圈但始终无法建立连接 
- 同一设备和线材在其他车辆或接口可用、在故障接口始终连不上 
- 需要晃动线束或插头才能偶尔接上、存在明显接触不良 
- 多个 USB/Type‑C 口同时断电，既不能连接数据也无法充电

【malfunction】 
定义：USB接口/Type‑C接口的故障类问题，指车内USB（含Type‑C）插口本身或其相关线路/模块异常，导致无法正常供电或进行数据传输（充电、识别U盘、CarPlay等功能失效），与误操作、使用方法无关。

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 某个或部分USB口完全无电，无法给手机充电 
- 插入U盘或行车记录仪无反应/无法被识别 
- USB口接触不良，需上抬或固定角度才有电 
- 同一位置双口一好一坏，或前排不能用但后排正常 
- 全车USB/Type‑C接口指示灯不亮，全部充电和数据功能失效 
- USB口松动、内部断裂或烧蚀导致长期无法使用 
- USB接口间歇性失效，充电/播放中途时有时无 
- 升级主机或更换模块后，相关USB口持续不可用 
- USB口异常致U盘、充电线发烫甚至烧熔 
- 媒体界面提示“USB系统暂时不可用/播放未启用”等功能故障提示

【stop charging】 
定义：USB 接口停止供电/充电的问题，指车内某个或全部 USB/Type‑C 充电口在应能给外部设备供电时，无法开始充电或充电过程中中断，表现为设备不进电或充电反复断开。 

特征性话题：（以下为常见示例，符合定义的反馈均应归入此类） 
- 某一个 USB/Type‑C 口不充电，同车其他口正常 
- 全车或前后排全部 USB 接口无电、指示灯不亮 
- 充电需“扶着/晃动线”才有电，松手立即断电 
- 充电几秒后自动中断或反复一会有电一会没电 
- 新车或使用不久即发现 USB 端口一直不充电 
- 同一条线在其他车辆/设备可充电，在本车 USB 口无反应

## 分类原则（重要！）
- **以定义为准**：判断反馈是否属于某个L2时，以该L2的"定义"为核心依据
- **特征性话题是示例**：每个L2下列出的"特征性话题"是该L2的常见情况示例，不是完整列表
- 只要反馈内容符合L2的定义描述，即使与列出的特征性话题不完全匹配，也应归入该L2
